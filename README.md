# Rapidito Delivery Application

## Features
- **Real-time Order Tracking**: Users can track their orders in real-time.
- **Multiple Payment Options**: Supports credit cards, debit cards, and digital wallets.
- **User-Friendly Interface**: Intuitive navigation and layout for a better user experience.
- **Push Notifications**: Receive instant updates on order status and promotions.
- **Multi-language Support**: Available in several languages for wider accessibility.

## Installation Instructions
1. **Clone the Repository**
   ```bash
   git clone https://github.com/cpisfilme01-gif/rapidito.git
   ```
2. **Navigate to Project Directory**
   ```bash
   cd rapidito
   ```
3. **Install Dependencies**
   ```bash
   npm install
   ```

## Usage Flows
### User Registration
1. Open the application.
2. Click on the "Register" button.
3. Fill in the required fields and submit.

### Placing an Order
1. Login to your account.
2. Browse the available items.
3. Add desired items to the cart.
4. Go to the cart and proceed to checkout.
5. Choose a payment method and confirm the order.

### Tracking an Order
1. Login to your account.
2. Navigate to the "My Orders" section.
3. Select the order you wish to track.
